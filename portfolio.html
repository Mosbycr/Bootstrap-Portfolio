The Egyptian Word Guess Game

Players use keys to guess randomly generated words. The current word is hidden. As letters are guessed they are filled in or placed in a seperate incorrect letters guessed area. Players have a set number of tries to guess the word and add a win to their score or lose and recieve another word.

Technologies Used: HTML5, CSS, Bootstrap4, JavaScript

https://mosbycr.github.io/Word-Guess-Game/

Crystal Collector Game

A game where the user tries to match a randomly generated number by selecting crystals with randomly assigned values until they reach the generated number and win or go over the generated number and lose.

Technologies Used: HTML5, CSS, Bootstrap4, JavaScript, jQuery

https://mosbycr.github.io/Crystal-Collector-Game/

Pirates Trivia Game

Allows the user to start a multiple choice question game about pirates. Users have 20 seconds to answer the question. Afterword they recieve the answer with various responses and gifs/images depending on if they were correct, incorrect, or did not answer before the timer lapsed. Once all questions have been answered users are shown their totals: correct, incorrect, and unanswered. At the result screen users may also restart the game by clicking the restart button.

Technologies Used: HTML5, CSS, Bootstrap4, JavaScript, jQuery, setTimer, setInterval

https://mosbycr.github.io/Pirates-Trivia-Game/

Superhero Giffy Fun

Users can add superhero buttons by adding names into the input box and hitting submit. Those buttons are then clicked causing an ajax request to search for 10 gifs of that superhero from Giffy Developer API. Those gifs are then displayed for the user's enjoyment. Users can animate the gifs by clicking on each one individually.

Technologies Used: HTML5, CSS, Bootstrap4, JavaScript, jQuery, AJAX, JSON APIs

https://mosbycr.github.io/Superhero-Giffy-Fun/

Train Scheduler

Allows users to view current running trains, their destination, frequency, next train, and time until arrival in minutes. Users can also add a train, it's destination, the first time the train runs and how often it runs. The program takes the users input and outputs the next train time and how long before it arrives. All entries are stored in firebase and are seen in the current trains data table.

Technologies Used: HTML5, CSS, Bootstrap4, JavaScript, jQuery, Moment.js

https://mosbycr.github.io/Train-Scheduler/

Daily Wheaties

A user creates or logs-in with the firebase built in UI to their personal account. The user then inputs information about their name, city and interests via a form to create a daily personalized application. This application shows an inspirational quote, a daily joke, personalized weather and a list of events for the day in the users area.

Technologies Used: HTML5, CSS, Bootstrap4, JavaScript, jQuery, AJAX, JSON APIs, Firebase built-in UI and database, Algolia Community Library

https://mosbycr.github.io/DailyWheaties/
